Neighbourly – Local Services Finder Web App

Neighbourly is web application that helps users discover nearby service professionals such as electricians, plumbers, cleaners, and tutors. The platform provides search, filter, and service detail views with a clean and professional UI.

This project is built as part of a frontend + backend using Flask and Vanilla JavaScript, without any database.

 Features:-

  Simple Login Page
  Search services by name
  Filter services by:
  * Category
  * Location
  * Availability (Available / Busy / Offline)
  * Service cards with:
  * Availability status
  * Rating
  * Verified professional tag
  * Dynamic service detail page
  * Fast loading using JSON-based data (no database)


 Tech Stack

 Frontend:-

* HTML5
* CSS3
* Vanilla JavaScript

 Backend:-

* Python
* Flask

 Data Handling

* JSON (used as a data source instead of a database)
